# Google Cybersecurity Certificate - Course 3

## Course Overview
Course 3 focuses on network architecture and security. Learners explore network-level vulnerabilities, protocols, tools, and techniques to secure systems and communications. The course introduces network hardening practices, OS security, and cloud network protection. Labs and exercises are included to reinforce hands-on learning.

## Modules

### Module 1 - Introduction to Networks
- Overview of networks, network tools, and components
- Cloud networks and software-defined networks
- Labs: Design a local area network, identify network tools
- Assessments: Introduction to networks quiz

### Module 2 - Network Protocols and Security Measures
- Common network protocols, wireless protocols, VPNs, firewalls, proxy servers
- Security zones, subnetting, and CIDR
- Labs: Explore and secure network protocols
- Assessments: Recognize network protocols quiz

### Module 3 - Network Attacks
- Denial of Service (DoS) attacks, packet sniffing, IP spoofing
- Tactics to secure compromised systems
- Labs: Analyze network layer communication, analyze network attacks
- Assessments: Secure networks against DoS attacks and network interception

### Module 4 - Network Hardening and Cloud Security
- OS hardening practices, brute force attack mitigation
- Network hardening, security applications, cloud hardening
- Labs: Apply OS hardening techniques, analyze network hardening, use NIST Cybersecurity Framework for incident response
- Assessments: OS hardening, network hardening, cloud security quizzes

## Portfolio and Notes
- Hands-on labs for network design and security
- Security hardening exercises
- NIST Cybersecurity Framework portfolio activity
