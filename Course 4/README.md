# Google Cybersecurity Certificate - Course 4

## Course Overview

Course 4 focuses on operating systems, Linux, and SQL. It builds the foundation needed to work with systems, manage files, and query data. The course includes hands-on labs to practice Linux commands, file management, user permissions, and SQL queries. All activities are documented as part of my portfolio.

## Modules

### Module 1 - Operating Systems

- Learned how operating systems work and their main functions
- Explored the relationship between hardware, software, and the OS
- Compared different operating systems
- Practiced understanding OS processes and resource management
- Assessments completed with 100%

### Module 2 - Introduction to Linux

- Learned Linux fundamentals and architecture
- Explored Linux distributions including Kali Linux
- Understood package managers and software installation
- Practiced working with the Linux shell and input/output
- Labs completed and documented in portfolio
- Assessments completed with 100%

### Module 3 - Linux Commands and File Management

- Navigated the file system using Bash commands
- Used commands to read, filter, and manage files
- Learned file permissions, ownership, and user management
- Practiced authentication and authorization in Linux
- Portfolio activity: managing file permissions using Linux
- Labs and exercises documented
- Assessments completed with 100%

### Module 4 - SQL and Databases

- Learned how databases work and how to query them using SQL
- Wrote basic queries and applied filters (WHERE, AND, OR, NOT)
- Worked with dates, numbers, and conditions
- Learned how to join multiple tables
- Portfolio activity: applying filters to SQL queries
- Labs documented in portfolio
- Assessments completed with 100%

## Portfolio and Notes

- Linux labs for file management, permissions, and user control
- Bash command practice and system navigation
- SQL query exercises including filtering and joins
- Portfolio activities demonstrating real-world use cases
