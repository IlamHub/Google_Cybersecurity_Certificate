# Google Cybersecurity Certificate - Course 5

## Course Overview

Course 5 focuses on asset security, covering how organizations identify, classify, and protect their assets. It explores risk management, security controls, cryptography, vulnerability management, and threat modeling. The course includes hands-on labs and portfolio activities that simulate real-world security scenarios.

## Modules

### Module 1 - Introduction to Asset Security

- Learned the what, why, and how of asset security
- Understood the relationship between risks, threats, and vulnerabilities
- Explored asset classification and common classification requirements
- Studied the elements of a security plan and the NIST Cybersecurity Framework
- Portfolio activity: classifying assets connected to a home network
- Portfolio activity: scoring risks based on likelihood and severity
- Assessments completed with 100%

### Module 2 - Protecting Organizational Assets

- Learned about security controls and the principle of least privilege
- Explored the data lifecycle and information privacy regulations (PII, GDPR, HIPAA, PCI DSS)
- Studied fundamentals of cryptography including symmetric and asymmetric encryption
- Understood public key infrastructure (PKI) and non-repudiation through hashing
- Explored access controls, authentication systems, SSO, MFA, and IAM
- Lab: decrypting an encrypted message
- Lab: creating and comparing hash values
- Portfolio activity: improving authentication, authorization, and accounting for a small business
- Assessments completed with 100%

### Module 3 - Vulnerability Management

- Learned the vulnerability management process and defense in depth strategy
- Studied common vulnerabilities and exposures (CVEs) and the OWASP Top 10
- Explored open source intelligence (OSINT) and vulnerability assessment approaches
- Understood penetration testing concepts and the importance of system updates
- Developed an attacker mindset by examining threat actors and attack vectors
- Studied brute force attacks and methods to fortify against them
- Portfolio activity: analyzing a vulnerable system for a small business
- Portfolio activity: identifying the attack vectors of a USB drive
- Assessments completed with 100%

### Module 4 - Threats to Asset Security

- Explored social engineering tactics and phishing types
- Learned to identify and filter malicious emails
- Studied malware types and the rise of cryptojacking
- Understood web-based exploits including XSS and SQL injection
- Learned the threat modeling process and applied the PASTA framework
- Applied PASTA to a real application threat model scenario
- Assessments completed with 100%

## Portfolio and Notes

- Asset classification activity for a home network
- Risk scoring matrix based on likelihood and severity
- Data handling practices and privacy compliance notes
- Cryptography labs covering encryption and hashing
- Authentication and authorization improvement for a small business
- Vulnerability analysis report for a small business system
- USB attack vector identification activity
- PASTA threat model framework applied to an application
